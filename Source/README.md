# Source Code

## PROWLER.BAS

The original BASIC source code used as the inspiration and initial example for
this program. Part of the "Hayes Hackamatic" found in various places around the
Internet (try Internet Archive).

## CHRONO.BAS

Sample code taken from the Hayes "Stack" Chronograph manual, it provides a
decent pattern for how to roll communications with the Super Serial Card 
attached device into a subroutine.

## HELLO.BAS

The HELLO program used on the Apple IIe diskette boot menu to start the program.

## TEST.BAS

Simple test program used for various things, typically to make sure things work
and build correctly.

## WARDIAL.BAS

The war dialer itself.
